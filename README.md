# SignalR Starter

Liten applikasjon som fungerer som starter template for ASP.Net Core med SignalR.

For frontend brukes Vue.js, hvor ASP.Net starter Vue CLI serveren for å gi oss HMR, etc.

## Prerequisites

* Visual Studio / Visual Studio Code
* NPM
* .Net Core 2.2 SDK

## Oppstart

1. Klon prosjekt
1. Åpne .csproj
1. Trykk start
